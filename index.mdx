SmartLogix is a full-stack logistics management platform built with Spring Boot microservices and a React dashboard. It covers the entire supply chain workflow — from registering inventory items to processing customer orders and tracking shipments through to delivery — all accessible through a single API gateway.

## Get started

<Steps>
  <Step title="Run the platform">
    Start all services with a single Docker Compose command. See the [quickstart](/quickstart) for the full setup guide.
  </Step>
  <Step title="Create an account">
    Register a user account via `POST /api/auth/register` to receive a JWT token for authenticated requests.
  </Step>
  <Step title="Add inventory">
    Use `POST /api/inventario` to register your first products with SKUs, warehouse locations, and stock levels.
  </Step>
  <Step title="Create an order">
    Submit `POST /api/pedidos` to place an order. SmartLogix validates stock in real time before confirming.
  </Step>
</Steps>

<Note>
  All API requests (except registration and login) require a `Authorization: Bearer <token>` header. See [Authentication](/features/authentication) for details.
</Note>
