SmartLogix is a full-stack logistics management platform built on a Spring Boot microservices backend and a React dashboard. It gives you end-to-end visibility over your supply chain — from tracking inventory across warehouses to processing customer orders and following shipments through to delivery — all behind a single API gateway secured with JWT authentication.

## Microservices architecture

SmartLogix runs as a set of independent services, each with its own database, coordinated through a central API gateway. All client traffic enters through the gateway on port `8080`; you never need to call individual services directly.

| Service                | Port   | Responsibility                               |
| ---------------------- | ------ | -------------------------------------------- |
| `api-gateway`          | `8080` | Routes requests and validates JWT tokens     |
| `auth-service`         | `8081` | User registration and login                  |
| `ms-inventario`        | `8082` | Inventory records, SKUs, and stock alerts     |
| `ms-pedidos`           | `8083` | Order creation and status management         |
| `ms-envios`            | `8084` | Shipment assignment and delivery tracking    |
| `notification-service` | `8085` | Async event-driven order approval alerts     |

## Technology stack

| Layer          | Technology                          |
| -------------- | ----------------------------------- |
| Backend        | Java 21, Spring Boot 3, Maven       |
| Frontend       | React 18, JavaScript (ES6+)         |
| Database       | PostgreSQL 15 (one instance per service) |
| Infrastructure | Docker, Docker Compose              |
